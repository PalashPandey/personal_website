Qlik Engine API Connector

Python host to interact with Qlik Sense enterprise. Uses web sockets to interface with the REST API. Created as a business intelligence developer at Thomas Jefferson University Hospital to automate Qlik application reloads, extracts, report downloads/ distribution etc. Published as a package on pypi as a pip installable package.

Summary

Qlik Engine API Connector lets a user connect to the Qlik Engine API directly using websockets and interact with their local Qlik server effortlessly. Users can automate report extracts with or without Nprinting, distribute pdf/ excel / image reports via email, monitor server usage by hours, check for idle users etc. To explore all the possibilities of the Engine API, go to engine API explorer under dev-hub on your Qlik installation.

Predatory Journal Tracker Chrome Extension

Google Chrome Extension to cross reference journal names from google scholar with Beall's list of predatory journals. This list would give the reader a warning about suspicious papers being passed off as legitimate research by highlighting suspicious papers in red on the google scholar's search page.

Summary

This list is in no way definitive but can give the users a waring sign for a research paper so they can investigate more. Frequently people use flimsy research papers online to peddle pseudo science and fake news. This is specially common in pseudo scientific medicine where people publish articles in "research journals" and make outlandish claims about medicinal benefits of "drinking cow urine".

STAR Scholars Program Project

This study analyzes gender bias in peer review by analyzing the sentiment of the reviewers comments on the submitted manuscripts.The study attempts to eliminate the quality of papers as an attribute in decision making process and thus compares similar papers to isolate effect of authors’ attributes like gender.

Summary

Publication is at the core of scientific communication and so the decisions made by conferences and journals are non trivial in the effect that they can have on the quality of the work that is being done and as a result the quality of work that gets shared with the world. We use peer read dataset of reviews and analyzing individual reviews that have been presented for manuscripts and analyzing their sentiment. We assign sentiment scores to the reviews and these scores act as a normalizing factor for the acceptance/rejection decisions so that the decisions can be analyzed with respect to authors’ gender without obfuscation from other factors.
